# PDF Password Protector

## Description

A Python-based PDF protection tool that encrypts PDF files with a password to prevent unauthorized access.

## Features

- Password-protect PDF files
- Secure PDF encryption
- User-defined passwords
- Command-line interface
- Automatic protected PDF generation

## Technologies Used

- Python 3
- PyPDF2

## Installation

```bash
git clone https://github.com/Vinaynagari795/Pdf-password-protector.git
cd Pdf-password-protector
```

## Requirements

```bash
pip3 install PyPDF2
```

## Usage

```bash
python3 pdf_password_protector.py input.pdf protected.pdf mypassword
```

Example:

```bash
python3 pdf_password_protector.py sample.pdf protected.pdf secure123
```
